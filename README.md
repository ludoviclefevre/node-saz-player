> A simple SAZ file player. Very early stage, changing often.


## Install

```sh
$ npm install --save saz-player
```


## Usage

```js
var saz-player = require('saz-player');

saz-player('Rainbow');
```

```sh
$ npm install --global saz-player
$ saz-player --help
```
